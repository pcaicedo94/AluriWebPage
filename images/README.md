# Images Folder

## Structure

- `/team/` - Team member photos

## Required Images

### Team Photos
Please add the following images to the `team` folder:

1. **oscar-zapata.jpg** - Photo of Oscar Zapata (CEO)
2. **oscar-tarazona.jpg** - Photo of Oscar Tarazona (CRO)
3. **jean-paul-manjarres.jpg** - Photo of Jean Paul Manjarres (CTO)

### Image Specifications
- **Format**: JPG or PNG
- **Aspect Ratio**: 1:1 (square)
- **Recommended Size**: 800x800px minimum
- **File Size**: Keep under 500KB for optimal loading

### How to Add Images

1. Save your team photos in the `images/team/` folder
2. Name them exactly as listed above
3. The website will automatically display them on the "Nosotros" page
4. If an image is missing, a gradient background will show as fallback

## Notes

- Images use lazy loading and hover effects
- Gradient backgrounds serve as fallbacks if images aren't available
- All images are optimized with object-cover for consistent display
